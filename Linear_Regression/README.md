# 📈 Simple Linear Regression

## 📌 Overview

Simple Linear Regression is a supervised machine learning algorithm used to predict a continuous target variable using a single independent feature.

It finds the best-fit straight line by minimizing the Sum of Squared Errors (SSE).

---

## 📊 Dataset

- **Dataset:** House Price Dataset (`houseprice.csv`)
- **Target Variable:** House Price
- **Features:** House-related attributes used for prediction.
